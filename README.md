# Love-Note

A simple web page that displays a heartfelt love note.

## Usage

To view the love note in your browser:

1. Start the local server:
   ```bash
   python3 -m http.server 8000
   ```

2. Open your browser and go to: http://localhost:8000

This will display the romantic message as a styled web page.

## Alternative: GitHub Pages

To host it online, enable GitHub Pages in your repository settings (under Pages, select main branch) and access it at: https://davinchi111-big.github.io/Love-Note/
